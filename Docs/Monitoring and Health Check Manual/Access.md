To view ModelOps Monitoring dasboard, follow these steps:

**Step 1**: Request QuickSight Access

Contact the QuickSight administrator to request access to Maximus QuickSight.

**Step 2**: Request ModelOps Monitoring Dashboard Access

Once QuickSight access is granted, reach out to the ModelOps dashboard administrator to request access to the ModelOps Monitoring dashboard.

**Viewing ModelOps Metrics**

After access is granted, follow these steps to log in and view the dashboard:

1. Login to Monitoring Dashboard

    1. Click [here](https://quicksight.aws.amazon.com/) to open the login page.
    2. Enter the credentials to sign in.


2. Navigate to Dashboards
    - Once logged in, locate the left-hand navigation pane.
    - Click on **Dashboards** to view the list of available dashboards.

3. Open the ModelOps Monitoring Dashboard
    - From the list of dashboards, select **ModelOps Monitoring dashboard**.
    - The dashboard will load and display various monitoring metrics.

!!!note
    - Ensure you have the necessary permissions to access the dashboard.
    - If you encounter access issues, contact the system administrator.
